# Phonics — Tedim Zolai

### Forbidden patterns
- No `ti` cluster — restructure or use `ci`
- No `c` before {a, e, o, aw} — use `k` or `kh`
- No pure /o/ — always /oʊ/

## Tones (3-way system)
| Tone | Mark | Example | Meaning |
|------|------|---------|---------|
| Level (mid) | none | `pai` | go |
| Rising | ´ | `pái` | carry |
| Falling | ` | `pài` | arrive |

Tone is phonemic — minimal pairs exist: `pai`/`pái`/`pài`

## Minimal pairs (key drills)
- `pai` (go) vs `bai` (white)
- `tui` (water) vs `dui` (push)
- `kum` (year) vs `gum` (hold)
- `lam` (road) vs `lham` (step)
- `pan` (board) vs `phan` (spread)
