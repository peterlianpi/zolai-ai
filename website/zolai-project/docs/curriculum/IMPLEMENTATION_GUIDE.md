## 📊 Content Creation Metrics

### Per Unit
- **Lessons**: 8 lessons
- **Topics**: 8 topics (1 per lesson)
- **Exercises**: 48 exercises (6 per lesson)
- **Audio**: 7-10 vocabulary words + examples
- **Time**: 2 weeks per unit

### Per Section
- **Units**: 4-5 units
- **Lessons**: 32-40 lessons
- **Exercises**: 192-240 exercises
- **Time**: 4-6 weeks per section

### Per Level (A1)
- **Sections**: 4 sections + daily refresh
- **Units**: 18 units
- **Lessons**: 144 lessons
- **Exercises**: 864 exercises
- **Time**: 16 weeks

### Phonics Track
- **Lessons**: 14 lessons (5 vowels + 6 consonants + 3 tones)
- **Exercises**: 140 exercises (10 per lesson)
- **Time**: 4 weeks
