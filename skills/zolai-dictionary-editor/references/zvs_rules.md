# ZVS (Zolai Vernacular Standard 2018) Rules

### Primary Mandate: Strict Tedim Zolai
This project is exclusively for the **Tedim** language. All non-Tedim (Hakha, Falam, Mizo, etc.) vocabulary, grammar, and orthography are **STRICTLY FORBIDDEN**.

### Linguistic Standards
1.  **Strict Tedim Vocabulary:**
    - ✅ `Pasian` (God) | ❌ `Pathian`
    - ✅ `Gam` (Land) | ❌ `Ram`
    - ✅ `Tapa` (Son) | ❌ `Fapa`
    - ✅ `Topa` (Lord) | ❌ `Bawipa`
    - ✅ `Kumpipa` (King/Ruler) | ❌ `Siangpahrang`
    - ✅ `Tua` / `Tua ciangin` (That/Then) | ❌ `Cu` / `Cun`
    - ✅ `Suah` (Born) | ❌ `Chuak`
    - ✅ `Suahtakna` (Freedom) | ❌ `Zalenna`
2.  **Plurality Logic:** Never use plural marker `uh` with first-person inclusive `i` (we).
    - *Correct:* `I pai hi` (We go)
    - *Wrong:* `I pai uh hi`
3.  **Phonetic Constraints:**
    - **No `ti`:** The `t` and `i` sounds never combine. Use `te` or alternate mappings.
    - **No `c` with `a, e, o, aw`:** Use `k` or `s` equivalents. `c` is only for `i` and `u` (e.g., `ci`, `cu`).
4.  **Apostrophe (Pawfi):** Correct placement is mandatory.
    - Contraction: `na'ng` (for `nading`), `k'ong`, `n'ong`.
    - Possession: `Pasian' thu` (God's word), `ka pu'`.
5.  **Stem Logic (Stem II Nominalization):** Use Stem II for all subordinates, purpose markers, and nominalizations.
    - `mu` -> `muhna`, `sep` -> `sepna`, `lian` -> `liatna`, `pai` -> `painading`.
6.  **Compound Orthography:** Compounds must be joined.
    - ✅ `nasep`, `leitung`, `vantung`, `nading`, `hihleh`.
    - ❌ `na sep`, `lei tung`, `van tung`, `na ding`, `hi leh`.
7.  **Negative Conditionals:** Strictly use `kei leh` or `kei a leh`.
    - ✅ `Nong pai kei leh...` | ❌ `Pai lo leh`

8.  **Vocabulary Standards for Food/Usage:**
    - **Tasty/Delicious:** Use ✅ `lim` | ❌ `tui` (means water/liquid)
    - **Fried:** Use ✅ `kan` (verb) or `kang` (result: fried/crispy/dried) | ❌ `thih` (means iron/die)
    - **Spicy/Hot:** Use ✅ `thak` | ❌ `thip` or `thik`
    - **Bitter:** Use ✅ `kha`
    - **Sweet:** Use ✅ `khum`
    - **Cooking Oil/Fat:** Use ✅ `sathau` | ❌ `thau` (too ambiguous)
    - **Rice Context:** 
        - `Buhtang` = Uncooked rice
        - `Buhsih` = Cooked rice
        - `An` = Meal/Food/Rice in general usage (e.g., `an tawh meh le'ng`)

9.  **Conditional Patterns:**
    - Use `leng` for singular/specific conditions (e.g., `dawn leng`, `pai leng`).
    - Use `lehang` for general/plural conditions.
    - Contraction `le'ng` is also valid for "if we".

10. **Real-Life Usage Examples:**
    - Prefer sentences that reflect actual habits: `Bepi kan pen an tawh meh le'ng lim mahmah hi.`
    - Use natural verbs like `sawp` (wash/clean) for rice: `Buhtang sawp inla huan in.`
