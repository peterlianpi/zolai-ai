# Zolai Pedagogical Curriculum (A1 - C2)

This directory contains the comprehensive, CEFR-aligned sentence pattern and grammar curriculum for the Zolai language. It is designed specifically to power the AI Language Tutor.

## Curriculum Foundation
The curriculum is built by synthesizing three primary sources:
1.  **ZVS 2018 Zolai Bible (66-Book Survey):** Provides massive, context-rich parallel data, ranging from basic Genesis narratives to complex Romans argumentation.
2.  **Zolai Sinna (Learning Manuals):** Provides the foundational structural rules and pedagogical progression.
3.  **Zolai Gelhmaanbu (Orthography & Style Guide):** Ensures all syntax and spelling align with formal standards (e.g., correct pluralization, negations, and pawfi usage).

## Proficiency Bands

The curriculum is divided into six progressive stages:

*   **[A1 Beginner](a1_beginner.md):** Core identity, existence, and simple SOV structures. (*Sinna Lessons 1–12*)
*   **[A2 Elementary](a2_elementary.md):** Sequential narrative, past tense markers, and basic prepositions. (*Sinna Lessons 13–22*)
*   **[B1 Intermediate](b1_intermediate.md):** Relational statements, reasoning (`Bang hang hiam cih leh`), and compound sentences. (*Sinna Lessons 23–30*)
*   **[B2 Upper-Intermediate](b2_upper_intermediate.md):** Complex logic, proper conditionals (`nong pai kei a leh`), and comparisons. (*Sinna Lessons 31–34*)
*   **[C1 Advanced](c1_advanced.md):** Rhetorical argumentation, abstract theology, and logical exposition.
*   **[C2 Mastery](c2_mastery.md):** Visionary metaphors, strict poetic parallelism, and high-register doxology.

## Usage for AI Tutors
When instructed to teach or correct a student, the AI Tutor should first determine the user's proficiency level, and then reference the corresponding curriculum file to generate level-appropriate sentences, vocabulary, and grammar rules. Ensure that you never exceed the user's level by more than one incremental step (+1 rule).
