## C1 — Section 1: Metaphor & Style

### Unit 1: Metaphorical Language
**"I Am" declarations (John pattern):**
- `Kei pen nuntakna an ka hi hi.` = I am the bread of life. (John 6:35)
- `Kei pen leitung khuavak ka hi hi.` = I am the light of the world. (John 8:12)
- `Kei pen lam, thuman, leh nuntakna ka hi hi.` = I am the way, the truth, and the life. (John 14:6)
- `Kei pen nuntakna leh thokiksatna ka hi hi.` = I am the resurrection and the life. (John 11:25)

**Pattern:** `Kei pen [abstract noun] ka hi hi.`
Double `hi hi` = absolute, eternal identity.

**Absolute declaration (Exodus 3:14):**
- `KEIMAH ka hi hi.` = I AM THAT I AM.
- Pattern: `[Emphatic pronoun] + [agreement marker] + hi hi`

### Unit 2: Stylistic Variation
**Register levels:**
| Register | Marker | Example |
|---|---|---|
| Intimate | Drop subject | `Pai in.` (Go.) |
| Formal | Full subject | `Nangmah pai in.` |
| Honorific | `topa` / `kumpipa` | `Topa aw, hong pia in.` |
| Literary | `pen` + double `hi` | `Amah pen Pasian ahi hi.` |
| Prophetic | `ci hi` + future | `Topa in hih bangin ci hi...` |

### Unit 3: Literary Devices
**Parallelism (A = B):**
```
Line A: Sihna aw, na zawhna koi lai-ah om hiam?
Line B: Sihna aw, na gu koi lai-ah a om hiam?
= O death, where is your sting? O grave, where is your victory?
```

**Chiasm (A-B-B-A):**
```
A: Pasian in eite hong it a,
B: eite mawhna hong lak ding
B: Tapa hong khal hi.
A: Eite in Pasian i itna hang hi lo hi.
```

**Anaphora (repeated opening):**
```
Bang hangin...
Bang hangin...
Bang hangin...
```

### Unit 4: Embedded Clauses II (Pauline)
**Stacked conditions before main verb:**
```
Sihna panin Jesuh a thokiksak Pasian' Kha Siangtho note sungah hong om leh,
sihna panin Khrih a hingkik sakpa mah in,
note sungah a om ama Kha Siangtho mah tawh,
a sithei pumpi a hong nungtasak ding hi.
```
= Romans 8:11 — four clauses, main verb at end.

**Rule:** In complex sentences, ALL subordinate clauses come before the main verb. The main verb is always last.

### Unit 5: Poetic Forms
**Lament structure (Job/Psalms):**
```
Opening: [Address] aw, [complaint]
Body: Bang hangin [question]?
      Bang hangin [question]?
Closing: [Trust statement] hi.
```

**Praise structure (Psalms):**
```
Call: [Imperative] un!
Reason: Bang hang hiam cih leh [reason].
Result: [Doxology] hi.
```

## C1 — Section 2: Advanced Grammar

### Unit 1: Complex Morphology
**Verb stem alternation:**
| Stem I (citation) | Stem II (nominalized) | Meaning |
|---|---|---|
| `dah` | `dahna` | put → putting |
| `lian` | `liatna` | go up → going up |
| `pai` | `paina` | go → going |
| `gen` | `genna` | say → saying |
| `bawl` | `bawlna` | do → doing |
| `thei` | `theihna` | know → knowing |
| `om` | `omna` | exist → existence |
| `nungta` | `nungtakna` | live → life |

**Rule:** Stem II used for: nominalization, purpose clauses, subordinate clauses.

### Unit 2: Aspect & Aktionsart
**Complete aspect system:**
| Marker | Aspect | Example |
|---|---|---|
| `ding hi` | prospective (future) | `pai ding hi` |
| `hi` | perfective (past) | `pai hi` |
| `khin` | perfect (completed) | `pai khin hi` |
| `zo` | completive (finished) | `a zo hi` |
| `ngei` | experiential (ever) | `pai ngei hi` |
| `nawn` | iterative (again) | `pai nawn hi` |
| `ta` | inchoative (change) | `om ta hi` |
| `pah` | punctual (sudden) | `om pah hi` |
| `lel` | continuative (still) | `om lel hi` |
| `thei` | potential (can) | `pai thei hi` |

### Unit 3: Evidentiality
**How Zolai marks information source:**
- Direct evidence: `ka mu hi` = I saw (it)
- Reported: `ci hi` = (they) said
- Inference: `a hi ding hi` = it must be
- Hearsay: `ci-in ka za hi` = I heard that...
- Quotative: `ci-in gen hi` = said saying

### Unit 4: Mirativity (Surprise/New Info)
**Markers of unexpected information:**
- `ta hi` = (it is) now so (surprise at change)
- `pah hi` = suddenly/immediately
- `en in` = behold! / look!
- `En un` = Look! (plural address)
- `lamdang` = amazing/surprising

### Unit 5: Discourse Cohesion
**Maintaining reference across sentences:**
- `amah` = he/she/it (maintains subject)
- `tua` = that (refers back to previous)
- `hih` = this (refers to current)
- `tua ahih ciangin` = therefore (logical consequence)
- `tua tham loin` = furthermore (addition)
- `ahih hangin` = however (contrast)

## C1 — Section 3: Bible & Literature

### Unit 1: Bible Corpus III — Epistles
**Pauline opening formula:**
`[Author] in [recipient] kiangah: Hehpihna leh nuamna na om hen.`
= From [Author] to [recipient]: Grace and peace be with you.

**Doxology formula:**
`[Subject] tungah minthan'na tawntung dingin om hen. Amen.`
= To [Subject] be glory forever. Amen.

**Benediction:**
`Topa Zeisu Khazih' hehpihna thupha ahi zongin, Pasian hong itna ahi zongin, Kha Siangtho kikhopna ahi zongin note khempeuh tawh om hen.`
= The grace of the Lord Jesus Christ, the love of God, and the fellowship of the Holy Spirit be with you all. (2 Cor 13:14)

### Unit 2: Hymn Corpus
**Hymn structure:**
- Verse: narrative/doctrinal content
- Chorus: repeated praise/response
- Bridge: emotional climax

**Common hymn vocabulary:**
| Zolai | English |
|---|---|
| la | song |
| la sak | sing |
| pahtawi | praise |
| minthan'na | glory |
| tawntung | forever/eternal |
| kilawm | worthy |
| thupha | grace/blessing |
| lungdam | joy/gladness |

### Unit 3: Prophetic Register
**Prophetic formula:**
`Topa in hih bangin ci hi:` = Thus says the Lord:
`Topa' kammal hong tung hi:` = The word of the Lord came:
`Topa in thu gen a:` = The Lord spoke:

**Prophetic future:**
`[event] ding hi` = [event] will happen
`[event] ding a kilawm hi` = [event] is fitting/worthy to happen

Nangmah in minthan'na, pahtawina, leh vangliatna

### Unit 3: Poetic Parallelism
**Hebrew-style parallelism (A = B):**
```
A: Ka Lungkhamna in tuipi bang lian a...
B: Ka Dahna in mual lianpi bangin hong neencip hi.
```
= My sorrow is as vast as the ocean... My grief crushes me like a great mountain.

**Synonymous parallelism:**
```
A: Sihna aw, na zawhna koi lai-ah om hiam?
B: Sihna aw, na gu koi lai-ah a om hiam?
```

**Antithetic parallelism:**
```
A: Eite in Pasian i itna hang hi lo hi,
B: Pasian in eite hong it a...
```

## C2 — Section 2: Mastery II

### Unit 1: Oratory & Rhetoric
**Rhetorical question patterns:**
- `Lungpil mi koi-ah a om hiam?` = Where is the wise man? (1 Cor 1:20)
- `Kua in note hong bawlsiat theih ding hiam?` = Who can harm you?
- `Kua in Pasian' itna panin eite a khen theih ding hiam?` = Who can separate us from God's love?

**Anaphora (repeated structure):**
```
Kua in...khen theih ding hiam?
Kua in...khen theih ding hiam?
Kua in...khen theih ding hiam?
```

### Unit 2: Extended Narrative
**Narrative sequence markers:**
| Zolai | Function |
|---|---|
| `A kipat cilin` | In the beginning |
| `Tua ciangin` | Then/After that |
| `Tua nung ciangin` | After that |
| `Ni tampi khit ciangin` | After many days |
| `A tawpna in` | Finally |
| `Tua ni-in` | On that day |
| `Khatvei` | Once/One time |

### Unit 3: Translation Mastery
**Translation principles:**
1. Meaning over word-for-word
2. ZVS dialect always (pasian, gam, topa)
3. SOV word order maintained
4. Directional particles (`hong`/`va`) reflect speaker perspective
5. Aspect markers match English tense context

**Common translation challenges:**
| English | Wrong | Correct |
|---|---|---|
| God loves us | `Pasian in kei te a lungdam hi` | `Pasian in eite hong it hi` |
| We went | `Kei te pai hi` | `I pai hi` |
| If you don't go | `Na pai kei a leh` | `Na pai kei a leh` |
| God is our Father | `pasian hi eite pa a hi` | `Pasian hi eite Pa a hi` |

## C2 — Section 4: Mastery IV

### Unit 1: Full Fluency Drills
**Speed translation (EN → ZO):**
1. God created the world. → `Pasian in leitung a piangsak hi.`
2. I am the way. → `Kei pen lam ka hi hi.`
3. Let there be light. → `Khuavak om hen.`
4. For God so loved the world. → `Bang hang hiam cih leh Pasian in leitung mite a it mahmah hi.`
5. I AM THAT I AM. → `KEIMAH ka hi hi.`

### Unit 2: Spontaneous Composition
**Prompts for C2 production:**
- Describe a vision using `ka khuadak leh...ka mu hi`
- Write a doxology using `kilawm hi` + `tawntung`
- Compose a lament using `bang hangin...hiam?` pattern
- Write a proverb using parallel structure (A = B)
- Translate a complex English passage maintaining SOV

### Unit 3: Peer Teaching
**Teaching another learner:**
- Explain ZVS rules in Zolai
- Correct errors using `[wrong] hi lo hi; [correct] ahi hi`
- Give examples from Bible corpus
- Use Socratic method: `Bangci ngaihsut na hi hiam?`
