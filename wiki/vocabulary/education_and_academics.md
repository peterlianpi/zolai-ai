# Education & Academics (Siamsinna le Lai lam thu)

## Concept / Rule
The Zomi have placed immense value on education since the introduction of the written script. The vocabulary for academics is formal and highly respected, often using the roots `Sia` (Teacher/Expert) and `Lai` (Letter/Book/Knowledge).

### Core Education Vocabulary
- **Siamsinna / Laisinna:** Education / Studying.
- **Sangsia:** Teacher (Sang = School, Sia = Teacher).
- **Sangnaupang:** Student / Pupil (School + Child).
- **Sang / Sanginn:** School / University.
- **Laibu:** Book.
- **Khangno:** Youth / Student-age population.
- **Teltawkpna / Theihsiamna:** Understanding / Comprehension.
- **Biakin Sang:** Sunday School / Religious Education.
- **Laibu Hawm:** Distributing books / Libraries.
- **Theihkholhna:** Exam / Test (Knowing beforehand / Assessment).
- **Phualpi / Bupi:** Headquarters / University main campus.
- **Kammal:** Word / Vocabulary.

## Decision / Application
The AI Tutor uses these terms explicitly when discussing learning strategies, grading, school scenarios, and pedagogical instructions. The AI itself adopts the persona of `Sangsia`.

## Reason
Providing a rich set of vocabulary for learning allows the AI to give accurate feedback using standard educational terminology. It enables the user to ask meta-questions about their own learning progress in Zolai.

## Pattern / Vocabulary Usage
When discussing education:
```zolai
Zolai laibu sim ding a thupi mahmah hi. (Reading Zolai books is very important.)
Sangnaupang te in laisin a theih ding-te kisam hi. (Students need to know their studies.)
Kei in Zolai sangsia ka hi hi. (I am a Zolai teacher.)
```

## Mistake / Anti-pattern
Do not conflate `Sia` (Teacher/Expert/Master) with `Siangtho` (Holy). While a religious teacher is often highly respected, the academic title `Sangsia` strictly means "School Teacher." Do not translate "exam" literally as "asking questions" without context; use `Theihkholhna` or `Tehna` for formal testing.
