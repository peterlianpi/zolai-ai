## Sentence Patterns

- `[Subject] [Verb] hi.` = [Subject] [Verb].
  - e.g. `lai thei hi.`
- `[Verb] ni.` = Let's [Verb].
  - e.g. `pai ni.`
- `[Subject] [Verb] kei hi.` = [Subject] does not [Verb].
  - e.g. `ken ka umkei hi.`
- `[Clause 1] aa leh [Clause 2].` = If [Clause 1], then [Clause 2]. / Because [Clause 1], [Clause 2].
  - e.g. `Aguai, Tuang in sang ah lai a hanciam mahmah aa leh tua laitak amah a mute in hih pa pen sangnaupang kician khatin ngaihsun uhhi.`
- `[Action] nading [Purpose/Need].` = In order to [Action], [Purpose/Need].
  - e.g. `dik takin malgawm pah theih nading Standard Format khat kisam ahi hi.`
