## Geography / Directions

| Zolai | English | Notes |
|---|---|---|
| vaang lam | north | vaang=north, lam=direction |
| vaang lam nung | south | nung=behind/opposite of vaang |
| zingsang lam | east | where sun rises — direction only, NOT "morning" |
| nitang lam | west | where sun sets — direction only, NOT "evening" |
| tuipi | ocean, sea | tui=water, pi=big |
| tuikhuk | port, harbor | tui=water, khuk=hole/bay |
| gamgi | border, boundary | gam=land, gi=edge |

**Correction notes:**
- `zingsang` = east (sun-rise direction). `zingsang` alone = morning. In direction context = east.
- `nitang` = west (sun-set direction). `nitang` alone = evening. In direction context = west.
- `nisatna lam` is NOT standard Tedim ZVS — do not use.

## Sentence Patterns (News Register)

### Reporting speech (quotative)
```
[Source] in: "[quote]," a ci hi.
IAEA makaipa in: "North Korea in thaupi lian bawlsak a siam zawzaw hi," a ci hi.
```

### Event happened on date
```
[Date] ni-in, [event] hi.
2026 kum April 15 ni-in, typhoon lianpi khat a om hi.
```

### Cause → result
```
[Cause] ahih manin [result] hi.
Huih hat ahih manin innkhum tampi a sia hi.
```

### Threat/warning pattern
```
[Actor] in [action] ding ci-in a dawt hi.
Tehran in a mun tuamtuamah hong do kik ding ci-in a dawt hi.
```
