## Verses

### v2
**EN:** Ye that stand in the house of the

**TDB77:** { Topa } phat un. I Pasian' inn huangsung, { Topa' } innsunga a ding, { Topa' } nasemte aw, { Topa } phat un; ama min phat un.

### v3
**EN:** Praise the

**TDB77:** Topa phat un, bang hang hiam cih leh Topa hoih hi; ama min phatna lasak un, bang hang hiam cih leh phat nop huai hi.

**ZVS:** { Topa } phat un; bang hang hiam cih leh { Topa } hoih hi. Ama min phatna la sak un; bang hang hiam cih leh amah phatnophuai hi.

**Grammar:** adjective: hoih (good)

### v4
**EN:** For the

**TDB77:** Bang hang hiam cih leh Topa in ama adingin Jakob teel a, ama neihsa dingin Israel teel khin ta hi.

**ZVS:** Bang hang hiam cih leh { Topa } in ama-a' dingin Jakob teel a, ama neihsa dingin Israel teel khinta hi.

**Grammar:** purpose: in order to | completed: already done

### v5
**EN:** For I know that the

**TDB77:** Bang hang hiam cih leh Topa lian hi, ci-in ka thei a, eite’ Topa pen pasian khempeuh sangin lianzaw hi.

**ZVS:** Bang hang hiam cih leh { Topa } lian hi, ci-in ka thei a, eite' { Topa } pen pasian dang khempeuh sangin a lianzaw hi.

**Grammar:** quotative: said/saying | serial verb: -in (and then)

### v6
**EN:** Whatsoever the

**TDB77:** Vantung leh leitung, tuipi leh a thukna sung khempeuhah Topa in ama ut peuhpeuh a bawl hi.

**ZVS:** Vantung leh leitung, tuipi leh a thukna sung khempeuhah { Topa } in ama ut peuhpeuh a bawl hi.

### v7
**EN:** He causeth the vapours to ascend from the ends of the earth; he maketh lightnings for the rain; he bringeth the wind out of his treasuries.

**TDB77:** Amah in leimong panin meiipi kai sakin, guahzuk nadingin khuaphialepte bawl a, ama saalte panin huih pusuak sak hi.

**ZVS:** Amah in leimong panin meiipi kaisakin, guahzukna dingin khuaphialepte bawl a, a huihkua panin huih a pusuaksak hi.

**Grammar:** purpose: in order to

**ZVS:** Amah in Egypt gamsungah mihing ganhing tacil khempeuh a vatmai hi.

### v9

**ZVS:** Minam tampite vatmai-in kumpi vangliante that a,
