## Verses

### v1
**EN:** Give unto the

**TDB77:** Topa aa ci un, van mite aw, minthan’na leh hatna Topa aa ci un.

**ZVS:** { Topa}-a' ci un, van mite aw; minthan'na leh hatna { Topa}-a' ci un.

### v2
**EN:** Give unto the

**TDB77:** Ama minthan’na Topa’ tungah pia un la, a siangtho kizepna tawh Topa bia un.

**ZVS:** Ama minthan'na { Topa' } tungah pia un la, a siangtho kizepna tawh { Topa } bia un.

**Grammar:** reciprocal: ki- prefix

### v3
**EN:** The voice of the

**TDB77:** Topa’ aw in tuite tungah om hi; a minthang Pasian ging damdam a, tui tampite tungah Topa om hi.

**ZVS:** { Topa' } awging tuipite tungah kiza hi. A minthang Pasian' aw ging damdam a, tui tampite tungah { Topa' } aw kiza hi.

### v4
**EN:** The voice of the

**TDB77:** Topa’ aw vanglian a, Topa’ aw in liatna tawh kidim hi.

**ZVS:** { Topa' } awging vanglian a, { Topa' } awging, liatna tawh kidim hi.

**Grammar:** reciprocal: ki- prefix

### v5
**EN:** The voice of the

**TDB77:** Topa’ aw in taamtaak singte kitansak a, Topa in Lebanon mual taamtaak singte kuaitan hi.

**ZVS:** { Topa' } awging in taamtak singte kitansak a, { Topa' } awging in Lebanon mual taamtak singte a kuaitan hi.

**Grammar:** reciprocal: ki- prefix

### v6
**EN:** He maketh them also to skip like a calf; Lebanon and Sirion like a young unicorn.

**TDB77:** Lebanon mual leh Sirion mualte bawngno leh gamsialno bangin amah in diangsak hi.

**ZVS:** Lebanon mual leh Sirion mualte bawngno leh gamsialno bangin amah in a diangsak hi.

### v7
**EN:** The voice of the

**TDB77:** Topa’ aw in meikuang taangkhia sak hi.

**ZVS:** { Topa' } awging in meikuang taangkhia sak hi.

### v8
**EN:** The voice of the

**TDB77:** Topa’ aw in gamlak lok a, Topa in Kadesh gamlak lok hi.

**ZVS:** { Topa' } awging in gamlak lok a, { Topa } in Kadesh gamlak lok hi.

### v9
**EN:** The voice of the

**TDB77:** Topa’ aw in singkungte kiheekgawp sak a, gammang tulak hawkkhia-in kolhsak hi; a biakinn sungah a vekpi un awng-in “Minthan’na” ci uh hi.

**ZVS:** { Topa' } awging in singkungte heekgawp a, gammang tulak hawkkhia-in a kolhsak hi. Ama biakinn sungah a vekpi-un awngin, “Pasian' min thang ta hen,” a ci uh hi.

**Grammar:** quotative: said/saying | serial verb: -in (and then) | reciprocal: ki- prefix | plural: uh marker

### v10
**EN:** The

**TDB77:** Tuicin tungah Topa tu hi; a tawntung kumpi hihna tawh Topa tu hi.

**ZVS:** { Topa } in tui thukpite uk a, a tawntung kumpi hihna tawh { Topa } a tu hi.

### v11
**EN:** The

**TDB77:** Topa in a mite hatna pia ta hen! Topa in a mite nopna thupha pia ta hen!

**ZVS:** { Topa } in ama mite tungah thahatna pia ta hen. { Topa } in ama mite tungah nopsakna thupha pia ta hen.
