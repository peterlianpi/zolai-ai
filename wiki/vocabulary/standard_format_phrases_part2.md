## Sentence Patterns

- `X (v) pen tang khat bekin i gelh leh khiatna tuam neih hi.` = X (verb) is written as one word and has a different meaning when written separately.
  - e.g. `Lua (v) pen tang khat bekin i gelh leh khiatna tuam neih hi.`
- `X (adv) pen a mai aa Verb/Adj tawh i mat leh tua X in a mai aa Verb/Adj pen Adverb suaksak hi.` = When X (adverb) is used before a Verb/Adjective, X causes the preceding Verb/Adjective to become an Adverb.
  - e.g. `Lua (adv) pen a mai aa Verb/Adj tawh i mat leh tua Lua in a mai aa Verb/Adj pen Adverb suaksak hi.`
- `Gtn. [Example sentence]` = Example:
  - e.g. `Gtn. Zu tam dawnluain lua hi.`
- `*X pen a mai aa Verb/Adj tawh mat hamtang ding.` = *X should be used before a Verb/Adjective.
  - e.g. `*Adverb pen a mai aa Verb/Adj tawh mat hamtang ding.`
- `X (Prefix) pen a mai aa Verb/Adj tawh i mat leh tua X in a mai aa Verb/Adj pen a lehlam suaksak hi.` = When X (Prefix) is used before a Verb/Adjective, X causes the preceding Verb/Adjective to become its opposite.
  - e.g. `Lo (Prefix) pen a mai aa Verb/Adj tawh i mat leh tua Lo in a mai aa Verb/Adj pen a lehlam suaksak hi.`
- `X (c.adj) pen a mai aa Verb/Adj tawh i mat leh tua X in a mai aa Verb/Adj pen uangzawsak hi.` = When X (comparative adjective) is used before a Verb/Adjective, X causes the preceding Verb/Adjective to increase/become more.
  - e.g. `Zaw (c.adj) pen a mai aa Verb/Adj tawh i mat leh tua Zaw in a mai aa Verb/Adj pen uangzawsak hi.`
- `X (conj) pen a mai aa Verb/Adj tawh i mat leh tua Ciang in a mai aa Verb/Adj pen pau kizom suaksak hi.` = When Ciang (conjunction) is used before a Verb/Adjective, it causes the preceding Verb/Adjective to connect words.
  - e.g. `Ciang (conj) pen a mai aa Verb/Adj tawh i mat leh tua Ciang in a mai aa Verb/Adj pen pau kizom suaksak hi.`
- `X (time) pen hun i genna ah malkhat in gawm ding hi.` = X (time) should be written as one word when referring to time.
  - e.g. `Ciang (time) pen hun i genna ah malkhat in gawm ding hi.`
- `X (v) pen a mai aa Verb/Adj tawh i mat leh tua X in a mai aa Verb/Adj pen a khiatna behlap hi.` = When X (verb) is used before a Verb/Adjective, X adds to the meaning of the preceding Verb/Adjective.
  - e.g. `Khawm (v) pen a mai aa Verb/Adj tawh i mat leh tua Khawm in a mai aa Verb/Adj pen a khiatna behlap hi.`
- `X = A tamzaw ah negative deihlohna lam ah kizangzaw hi.` = X = Mostly used in the negative sense of not wanting something.
  - e.g. `Lua = A tamzaw ah negative deihlohna lam ah kizangzaw hi.`
- `X = A tamzaw ah positive deihna, hoihsakna lam ah kizangzaw hi.` = X = Mostly used in the positive sense of desire or approval.
  - e.g. `Mahmah = A tamzaw ah positive deihna, hoihsakna lam ah kizangzaw hi.`
- `X = Thugen pen in mi dangte a bulphuhna ah zangh hi.` = X = Used by the speaker to refer to other people.
  - e.g. `Hong = Thugen pen in mi dangte a bulphuhna ah zangh hi.`
- `X = Thugen pen in a mah le a mah a kibulphuhna ah zangh hi.` = X = Used by the speaker to refer to themselves.
  - e.g. `Ong = Thugen pen in a mah le a mah a kibulphuhna ah zangh hi.`
- `X = Ahiding, ahi nailote ah kizangh hi.` = X = Used for things that will happen or have not yet happened (future).
  - e.g. `Hing = Ahi ding, ahi nailote ah kizangh hi.`
- `X = Ahita, ahi laitakte ah kizangh hi.` = X = Used for things that have happened or are happening now (present/past).
  - e.g. `Ing = Ahita, ahi laitakte ah kizangh hi.`
- `X = Khat kia genna ah kizangh hi.` = X = Used when referring to one item/person.
  - e.g. `Leng = Khat kia genna ah kizangh hi.`
- `X = Khat sangin a tamzaw genna ah kizangh hi.` = X = Used when referring to more than one item/person.
  - e.g. `Lehang = Khat sangin a tamzaw genna ah kizangh hi.`
- `X = Ahi thei, ahilo theite ah kizangh hi. (English ah “if”)` = X = Used for possibilities or uncertainties (English 'if').
  - e.g. `Leh = Ahi thei, ahilo theite ah kizangh hi.`
- `X = khat le khat kikal gennate ah kizangh hi. (English ah “and”)` = X = Used when connecting one thing to another (English 'and').
  - e.g. `Le = khat le khat kikal gennate ah kizangh hi.`
- `X = Ei le ei i kigenna ah khat sangin i tamzawk leh kizangh hi.` = X = Used when the speakers refer to themselves as a group (more than one).
  - e.g. `I/Ei = Ei le ei i kigenna ah khat sangin i tamzawk leh kizangh hi.`
- `X = Mi dangte ahi zong, ei le ei ahi zong i genna ah khat sangin a tamzawk leh kizangh hi.` = X = Used when referring to a group (more than one), whether it's others or ourselves.
  - e.g. `Uh = Mi dangte ahi zong, ei le ei ahi zong i genna ah khat sangin a tamzawk leh kizangh hi.`
- `X = Khat bek tung ah genna` = X = Used when referring to one person/thing.
  - e.g. `In = Khat bek tung ah genna`
- `X = Khat sangin a tamzaw tung ah genna` = X = Used when referring to more than one person/thing.
  - e.g. `Un = Khat sangin a tamzaw tung ah genna`
- `Noun/Verb + In = A sem a bawl /a hih ding khatpeuh in a gen hun ciang “in” pen kituam gelh hi.` = Noun/Verb + In = When referring to an action being done or about to be done, 'in' is written as one word.
  - e.g. `Naupangte in an ne nuamta uhhi, an na pia in.`
- `Noun/Verb + In = Kammal kizomin zatnate ah a mai aa laimal tawh kimat hi.` = Noun/Verb + In = Used when combined with words, attached to the preceding word.
  - e.g. `Tuang in an a nek laitakin puan na sawp in.`
- `X = Kampau ciang “X” cih bangin kiza kha thei aa, tua bangin i gelh leh paulai hi-in, laimal dik taktak hilo hi.` = X = In spoken language, it sounds like 'X', but when written, it is written as 'Y' and is not the actual word.
  - e.g. `Se = Kampau ciang “se” cih bangin kiza kha thei aa, tua bangin i gelh leh paulai hi-in, laimal dik taktak hilo hi.`
- `X = Lai-in i gelh ciang “X” mah ci in kigelh tektek hi.` = X = When written formally, it is written exactly as 'X'.
  - e.g. `Sia = Lai-in i gelh ciang “sia” mah ci in kigelh tektek hi.`
- `A = Noun ii mai ah ahih leh laimal halin “a” mal khat bek zat ding ahi hi.` = A = Used as a single word 'a' before a Noun or as a word divider.
  - e.g. `A = Noun ii mai ah ahih leh laimal halin “a” mal khat bek zat ding ahi hi.`
- `Aa = Noun ii nung ah ahih leh laimal halin “aa” mal nih zat ding ahi hi.` = Aa = Used as two words 'aa' after a Noun or as a word divider.
  - e.g. `Aa = Noun ii nung ah ahih leh laimal halin “aa” mal nih zat ding ahi hi.`
