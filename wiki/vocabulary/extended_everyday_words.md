# Extended Everyday Vocabulary
## Concept / Rule
While our core vocabulary covers fundamental concepts, theology, and nature, everyday conversational fluency requires a broader set of practical words. These words cover household items, daily actions, food, and common states of being. They are essential for communicating daily needs, describing routines, and engaging in casual interactions.

## Decision / Application
Use these words when generating daily conversation scenarios, reading practice, or situational dialogues (e.g., shopping, eating, moving around the house). They should be introduced to learners transitioning from beginner to intermediate levels, moving beyond simple object identification to describing actions and states.

## Reason
Providing a richer vocabulary of everyday items and actions allows for more dynamic and realistic sentence construction. It bridges the gap between structured grammar exercises and practical, real-world communication.

## Pattern / Vocabulary List

### Actions & Verbs (Daily Routine)
- `tho` - wake up
- `lum` / `ihmu` - sleep
- `ne` / `an ne` - eat
- `dawn` / `tui dawn` - drink
- `gen` / `pau` - say / speak
- `ngaihsun` - think
- `panpih` / `hu` / `khop` - help
- `khawl` / `tang` - stop
- `pan` - start / begin
- `tawp` / `manne` - finish / end
- `tai` / `zom` - run

### Food & Kitchen
- `buh` / `an` - rice
- `meh tui` - soup
- `ci` - salt
- `anteh` - vegetable
- `ngahpi` - banana
- `khak` / `thuk` / `kha` - sour
- `khum` - sweet
- `thak` / `sa` - spicy

### Household & Personal Items
- `kongkhak` - door
- `lupna` - bed
- `touna` - chair
- `sabawi` - table
- `puan` - clothes
- `khedap` - shoe
- `tem` - knife
- `kuang` - spoon / plate
- `bel` - pot
- `hai` - cup

### Common Adjectives & States
- `gim` - tired
- `gialzawng` / `gil kial` - hungry
- `dangtak` - thirsty
- `hoih` - good
- `sia` - bad
- `manlang` / `hat` - fast / quick
- `nuam` / `khawl` - slow
- `haksa` - hard / difficult
- `nuam` / `baih` - easy
- `gik` - heavy
- `zang` - light (weight)

### Time & Sequence
- `tuni` - today
- `zingciang` - tomorrow
- `zani` - yesterday
- `zinglam` / `zing` - morning
- `nitaklam` / `nitak` - evening
- `tu` / `tu-in` - now
- `naciang` / `khit nung` - later

## Mistake / Anti-pattern
Do not use archaic or formal vocabulary when a simple everyday term suffices for casual conversation. For example, use `an ne` (eat) rather than formal feasting terminology when describing a typical family dinner. Ensure the context matches the level of formality.
