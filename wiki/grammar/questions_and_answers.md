## Question Structures (Summary)

| Pattern | Meaning | Example |
|---|---|---|
| `[verb] hiam?` | Did/does/will...? | `Na pai hiam?` = Did you go? |
| `[verb] khin hiam?` | Have you (already)...? | `Na pai khin hiam?` = Have you gone? / Did you go? |
| `[verb] ngei hiam?` | Have you ever...? | `Na mu ngei hiam?` = Have you ever seen (it)? |
| `ahi hiam?` | Is it...? | `Pasian kua hiam?` = Who is God? |
| `ding hiam?` | Will...? | `Hong pai ding hiam?` = Will (he) come? |
| `om hiam?` | Is there...? | `Koi lai-ah om hiam?` = Where is it? |
| `thei hiam?` | Can...? | `Na thei hiam?` = Do you know? |
| `bang...hiam?` | What...? | `Bang ahi hiam?` = What is it? |
| `bangci...hiam?` | How...? | `Bangci na hi hiam?` = How are you? |
| `kua...hiam?` | Who...? | `Kua in...hiam?` = Who...? |
| `bang hangin...hiam?` | Why...? | `Bang hangin tua bangin...hiam?` |
| `koi lai-ah...hiam?` | Where...? | `Koi lai-ah om hiam?` = Where is it? |

### Past tense questions

| English | Zolai | Notes |
|---|---|---|
| Did you go? | `Na pai hiam?` | simple past question |
| Did you go? (completed) | `Na pai khin hiam?` | `khin` = already done |
| Have you ever seen it? | `Na mu ngei hiam?` | `ngei` = ever/before |
| Did you not go? | `Na pai kei hiam?` | negated past |
| Have you already eaten? | `An na ne khin hiam?` | `khin` = completed |
