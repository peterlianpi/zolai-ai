## Agreement Marker System

Zolai uses **agreement prefixes** on verbs (not pronouns) to mark person.

| Person | Prefix | Example | Meaning |
|---|---|---|---|
| 1st singular | `ka` | `ka pai hi` | I go |
| 2nd singular | `na` | `na pai hi` | you go |
| 3rd singular | `a` | `a pai hi` | he/she/it goes |
| 1st plural | `i` | `i pai hi` | we go |
| 2nd plural | `na...uh` | `na pai uh hi` | you (all) go |
| 3rd plural | `a...uh` | `a pai uh hi` | they go |

**Critical rule:** `uh` = plural agreement marker for 2nd and 3rd person plural. NEVER use `uh` with `i` (1st plural).

## 1st Person Plural: `i` vs `ka uh`

- `i` = we (subject prefix, 1st plural inclusive) — 49 occurrences
- `ka uh` = we (1st plural, rare variant) — only 1 occurrence
- **Standard form:** `i pai hi` = we go/went

**Corpus evidence:**
- `I pai hi.` = We went. ✅
- `I om hi.` = We are. ✅
- `I bawl hi.` = We did it. ✅
- `I pai uh hi.` = ❌ WRONG — never add `uh` with `i`

## Nominalization: `-na` Suffix (Stem II)

Adding `-na` to a verb creates a noun (action → concept):

| Verb (Stem I) | Noun (Stem II) | Meaning |
|---|---|---|
| `biak` | `biakna` (1,538) | worship → worship/prayer |
| `mawh` | `mawhna` (702) | sin → sin/fault |
| `hehpih` | `hehpihna` (378) | be gracious → grace |
| `gim` | `gimna` (378) | suffer → suffering/trouble |
| `nungta` | `nungtakna` (374) | live → life |
| `khial` | `khialhna` (351) | err → error/sin |
| `lungdam` | `lungdamna` (330) | be glad → joy/gladness |
| `heh` | `hehna` (315) | be angry → anger |
| `siat` | `siatna` (305) | destroy → destruction |
| `ngeina` | `ngeina` (304) | custom → custom/habit |
| `pil` | `pilna` (302) | be wise → wisdom |
| `vangliat` | `vangliatna` (297) | be mighty → power/might |
| `si` | `sihna` (296) | die → death |
| `kikhop` | `kikhopna` (275) | gather → gathering/assembly |
| `gamtat` | `gamtatna` (270) | do/act → action/deed |
| `pai` | `paina` | go → going/journey |
| `gen` | `genna` | say → saying/speech |
| `thei` | `theihna` | know → knowledge |
| `om` | `omna` | exist → existence/dwelling |

**Rule:** Use Stem II (`-na`) for:
- Nominalization: `paina` = the going / journey
- Purpose clauses: `pai nadingin` = in order to go
- Subordinate clauses: `a pai ciangin` = when he went

## Common Agreement Marker Patterns

### 1st person singular (`ka`)
```
ka pai hi     = I go/went
ka thei hi    = I know/can
ka nei hi     = I have
ka om hi      = I am/exist
ka gen hi     = I say/said
ka it hi      = I love
ka lum hi     = I sleep
ka hei hi     = I am thirsty
ka ngah hi    = I am hungry
ka dam hi     = I am well
```

### 2nd person singular (`na`)
```
na pai hi     = you go/went
na thei hiam? = do you know?
na dam hiam?  = are you well?
na gen hi     = you say/said
na om hi      = you are/exist
```

### 3rd person singular (`a`)
```
a pai hi      = he/she goes/went  (2,994 occurrences of "a ci")
a gen hi      = he/she says/said
a nei hi      = he/she has
a bawl hi     = he/she does/did
a tun hi      = he/she arrives/arrived
a si hi       = he/she dies/died
a om hi       = he/she is/exists
```

### 1st person plural (`i`)
```
i pai hi      = we go/went
i om hi       = we are
i bawl hi     = we do/did
i it hi       = we love
i thei hi     = we know
```

### 3rd person plural (`a...uh`)
```
a pai uh hi   = they go/went
a gen uh hi   = they say/said
a om uh hi    = they are/exist
a bawl uh hi  = they do/did
```

## They vs We — Critical Distinction

| English | Correct Zolai | Wrong |
|---|---|---|

| We go | `I pai hi` | `I pai uh hi` ❌ |
| They said | `Amaute ci uh hi` | — |
| We said | `Kote ci uh hi` / `I gen hi` | — |
| They came | `Amaute hong pai uh hi` | — |
