# Grammar

Tedim Zolai grammar rules, patterns, and reference materials following the ZVS standard.

## ZVS Dialect Rules (Summary)

- Use: `pasian`, `gam`, `tapa`, `topa`, `kumpipa`, `tua`
- Never: `pasian`, `gam`, `tapa`, `topa`, `???`, `tua/tuan`
- Word order: **SOV** (Subject-Object-Verb)
- Negation: `kei` not `lo` for conditionals
- Plural: never combine `uh` with `i` (we)
- `o` is always /oʊ/ — never pure /o/

See individual files for detailed rules on each grammatical category.
