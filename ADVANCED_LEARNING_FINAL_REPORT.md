## Verified Grammar Rules

### Agreement Prefixes
```
ka = 1st person singular
na = 2nd person singular
a = 3rd person singular
i = 1st person plural
eite = 1st person plural exclusive
```

### Directional Particles
```
hong = toward speaker (recipient)
va = away from speaker
```

### Ergative Marker
```
in = marks agent/subject only
```

### Negation
```
kei = standard marker (ZVS)
kei a leh = conditional negation
lo = deprecated (avoid)
```

### Plural Marking
```
uh = 3rd person plural
i = 1st person plural
❌ Never combine i + uh
```

## Discussion Group Findings

### Consensus Rulings (200 discussions)

**Agreement Prefixes:**
- ✅ 3rd person + 'in' + recipient agreement + directional
- ✅ 1st person uses 'ka' directly
- ✅ Never mix agreement prefixes

**Directional Particles:**
- ✅ 'hong' mandatory for recipient marking
- ✅ 'va' for away from speaker
- ✅ Optional when 3rd→3rd

**Negation:**
- ✅ Use 'kei' (not 'lo')
- ✅ 'kei a leh' for conditional
- ✅ Never use 'lo leh'

**Plural Marking:**
- ✅ 'uh' for 3rd person plural
- ✅ 'i' for 1st person plural
- ✅ Never combine both

**Ergative Marker:**
- ✅ 'in' on agent only
- ✅ Never on object
- ✅ Required for 3rd person agents
