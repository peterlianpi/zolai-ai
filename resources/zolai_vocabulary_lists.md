# Zolai Vocabulary Lists
## Comprehensive Dictionary-Based Reference

This file is a comprehensive vocabulary reference built from the Tongdot Dictionary (21,847 entries across Tedim, Haka, Laizo/Falam, Matu dialects) combined with existing repo material.

**Sources:**
- `data/tongdot_dictionary.jsonl` — 27,758 dictionary entries
- `data/zolai_core_vocabulary.json` — 21,847 filtered Zolai/Chin entries
- `data/zolai_complete_vocabulary.json` — 12,521 unique Tedim words
- `resources/zolai_grammar_cheat_sheet.md`
- `src/zolai/v9/standardizer.py`
- `references/zolai-sinna-markitdown.md`

**Status legend:**
- `Confirmed`: directly supported by repo references or standardization rules
- `Attested`: form is attested in dictionary/corpus sources
- `Loanword`: modern borrowing used directly in Zolai
- `Dialect`: word from related Chin dialect (Haka, Laizo/Falam, Matu)

## Extended Dictionary (All Tedim Words)

The complete Tedim dictionary contains **12,521 unique words**. Key additions from the Tongdot Dictionary beyond the core vocabulary above include:

### Loanwords & Modern Terms
- **English loanwords**: Computer, Phone, Internet, Email, App, Data, Software, Program, Website, Mobile
- **Cultural loanwords**: Hospital, Bazaar, Christmas, Easter, Buddhist, Christian, Anglican
- **Brand names**: Coca-Cola (dawntheih tuikhum)

### Specialized Vocabulary
- **Legal terms**: Dan (law), thukham (commandment), thuman (truth)
- **Medical terms**: Dam (healthy), nat (disease), damsak (heal)
- **Agricultural terms**: Suan (garden), bu (grain), hum (fish), sa (meat)
- **Musical terms**: La (song), sa (sing), buan (read aloud)

### Compound Patterns
Common compound formation patterns found in dictionary:
- **Noun + Noun**: innkuan (inn + kuan = family), laibu (lai + bu = book)
- **Verb + na**: itna (it + na = love), pilna (pil + na = wisdom)
- **Adjective + Noun**: mihing (mi + hing = human), siangtho (siang + tho = holy)
- **Prefix + Root**: kipawlna (ki + pawl + na = organization)
